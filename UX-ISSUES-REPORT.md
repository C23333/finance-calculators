# UX优化问题报告

## 用户反馈

用户报告：
1. **样式还是很奇怪** - 布局或视觉效果有问题
2. **i18n适配还是不完美** - 国际化翻译不完整或显示有问题

## 需要检查的问题

### 1. 样式问题

可能的原因：
- CSS变量未定义（`--bg`, `--text`, `--primary`等）
- 响应式断点不合适
- 字体大小太小
- 间距不合理
- 颜色对比度不够

**需要检查：**
- [ ] `css/style.css` 是否定义了所有CSS变量
- [ ] 深色模式是否正常工作
- [ ] 移动端布局是否正确
- [ ] 字体大小是否合适（特别是中文）
- [ ] 按钮和链接的样式是否清晰

### 2. i18n问题

可能的原因：
- 中文翻译不完整
- data-i18n属性缺失
- i18n.js未正确加载
- 翻译键不匹配

**需要检查：**
- [ ] 所有UI文本是否有data-i18n属性
- [ ] zh.json是否包含所有翻译键
- [ ] 侧边栏内容是否有i18n支持
- [ ] 动态生成的内容是否翻译

## 具体需要修复的内容

### 侧边栏内容i18n

侧边栏中的教育内容（How to Use, Tips, FAQ）目前是硬编码的英文，需要：
1. 为所有侧边栏内容添加data-i18n属性
2. 在zh.json中添加对应的翻译
3. 确保i18n.js能正确处理侧边栏内容

### CSS变量检查

需要确保以下CSS变量在style.css中定义：
```css
--bg: 背景色
--bg-soft: 柔和背景色
--bg-muted: 静音背景色
--text: 主文本色
--text-secondary: 次要文本色
--text-light: 浅文本色
--primary: 主色
--primary-dark: 深主色
--primary-bg: 主色背景
--secondary: 次要色
--border: 边框色
--border-light: 浅边框色
--radius: 圆角
--radius-sm: 小圆角
--radius-full: 完全圆角
--transition: 过渡效果
```

### 字体大小优化

中文字体通常需要比英文稍大，建议：
- 标题：增加10-15%
- 正文：增加5-10%
- 小字：保持或略微增加
