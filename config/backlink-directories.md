# Backlink & Directory Submission Guide

Submit FinCalc to these directories and platforms to build quality backlinks.

## Free Directory Submissions

### General Directories
- **Product Hunt**: https://www.producthunt.com - Launch as "Free Financial Calculators"
- **AlternativeTo**: https://alternativeto.net - Add as alternative to NerdWallet, Bankrate calculators
- **SaaSHub**: https://www.saashub.com - Finance tools category
- **ToolFinder**: https://toolfinder.co

### Finance-Specific
- **Investopedia**: Tools directory (if they accept submissions)
- **NerdWallet**: Partner/affiliate program for calculator tools
- **Bankrate**: Submit calculator tools
- **WalletHub**: Financial tools directory

### Developer/Tech
- **GitHub**: Add link in README of any related open-source projects
- **Dev.to**: Write article "Building a Financial Calculator with Vanilla JS"
- **Medium**: Publish "How to Use Financial Calculators for Budget Planning"
- **Hashnode**: Developer-focused finance tools article

## Content Marketing (Guest Posts & Tutorials)

### Targets
1. **Personal finance blogs** - Offer guest post with calculator embed/link
2. **Real estate sites** - Mortgage/rent vs buy calculator content
3. **Investing education** - Retirement/401k calculator tutorials

### Platforms to Publish
- **Medium** - Use "Finance" and "Personal Finance" tags
- **Dev.to** - Technical implementation articles
- **Substack** - Finance newsletter mentions

## Community Participation

### Reddit
- r/personalfinance - Answer questions, mention calculators when relevant
- r/investing - Share educational content
- r/financialindependence - FIRE calculator content
- **Rules**: Add value first, no link dropping in posts, use flair

### Quora
- Answer "How much house can I afford?" type questions
- Link to relevant calculator in "Sources" section

### Hacker News
- "Show HN: Free financial calculators - mortgage, retirement, debt payoff"
- Post when you have meaningful updates

### Indie Hackers
- Share revenue journey with AdSense
- "Building a finance calculator site" story
